Spritesheet to SVG Animator Tool - v2.0

Instructions

  1. Select your input method (JSON File, Manual Inputs, or Custom JSON Builder). Each method has its own dedicated configuration area.
  2. Upload your sprite sheet PNG and, if needed, an Aseprite JSON file or fill in manual frames. Alternatively, build custom JSON via the builder modal.
  3. Adjust animation settings (scaling, layout, loop count, total duration, and compression level).
  4. Generate the SVG to see a live preview; then copy or download the final code or Data URI.

Tip: Always test your final SVG across different browsers and devices before deploying on‑chain.
